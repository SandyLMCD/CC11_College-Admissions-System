## Workflow / Functionalities

This one is just for one student/user who wants to enroll in the educational institution.
1. User should be able to register themselves if they haven't registered yet.
2. The program will evaluate the User's GPA and entrance exam score to see if they qualify for admission.
3. User can select the college they want to get into. In this program, there are three choices namely; College of Engineering, College of Computer Studies, and College of Nursing.
4. User can select the subjects they want to enroll in.
